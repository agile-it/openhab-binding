The binding is ${authorisationStatus}.

In order to authorise the binding:

  1. Create an instance of the Viessmann API Bridge Thing
  2. Go to the Viessmann Developer Portal and register for a developer account then add a client with the following redirect URI: ${redirectUri}

  3. Configure the Viessmann API Bridge with the client ID from the Viessmann Developer Portal

Once this is done, then click here

If authorisation is successful then the binding should automatically discover any attached heating devices and they will appear in your OpenHAB Inbox
